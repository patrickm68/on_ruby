h1. Hamburg on Ruby "!http://travis-ci.org/phoet/hamburg_on_ruby.png!":https://secure.travis-ci.org/phoet/hamburg_on_ruby.png

Source for the "Site of Hamburg's Ruby Community":http://onruby.de.

h2. Setup

* install rvm
* install rvm-version of .rvmrc
* brew install coffee-script

h3. GitHub-Authentication

You need "GitHub App credentials":https://dev.twitter.com in order to have a working login.
See config/initializers/omniauth.rb for details.

h2. Testing

The best way to execute tests is via calling @rake@ or @guard@

h2. Running

<pre><code>bundle --without=production # don't want to install postgresql stuff...
foreman start
</code></pre>

h2. Heroku for Rails 3.1

Heroku is not supporting the asset-pipeline properly at the moment, so you need to checkin the precompiled stuff...

<pre><code>rake assets:precompile
git add .
git commit -m 'precompiled'
git push heroku master
</code></pre>
